# 5维度审核清单与评分标准

## 1. 通顺度 (fluency)

### 评分标准
| 分数 | 标准 |
|------|------|
| 9-10 | 文章完全通顺，行文流畅，长难句逻辑清晰，无任何语法错误 |
| 7-8 | 文章基本通顺，偶有轻微生硬，但不影响理解 |
| 5-6 | 有明显语法错误或表达不自然，影响理解 |
| 1-4 | 文章不通顺，多处语法错误，无法正常阅读 |

### 审核要点
- [ ] 全文无语法错误
- [ ] 长难句逻辑连贯，不是生硬拼接
- [ ] 段落间有衔接手段
- [ ] 用词准确，无中式英语
- [ ] 无口语化表达

### 常见扣分项
- 段落间缺乏过渡（-1）
- 长难句逻辑混乱（-2）
- 中式英语表达（-1）
- 语法错误（每处-1）

---

## 2. 答案唯一性 (unique_correct_answer)

### 评分标准
| 分数 | 标准 |
|------|------|
| 10 | 4题答案均唯一确定，无任何争议 |
| 8-9 | 4题答案基本唯一，个别选项有微小歧义但不影响判断 |
| 5-7 | 有1-2题存在明显歧义，可能有两个"正确"答案 |
| 1-4 | 多题答案有争议 |

### 审核要点
- [ ] 每题恰好1个正确答案
- [ ] 正确答案在原文中有明确依据
- [ ] 错误选项与正确答案有明确区分
- [ ] 不存在"两个都对"的情况

### 常见扣分项
- 推理题的推断过度，答案不唯一（-2）
- 细节题有两个选项都"在文中提到"（-2）
- 主旨题选项区分度不够（-1）

---

## 3. 解析说服力 (explanation_persuasiveness)

### 评分标准
| 分数 | 标准 |
|------|------|
| 9-10 | 推理链精准引用原文，排除链逐一分析，考点概括准确 |
| 7-8 | 推理链和排除链基本完整，偶有不够具体的地方 |
| 5-6 | 推理链缺少原文引用，或排除链遗漏选项 |
| 1-4 | 解析空洞，无说服力 |

### 审核要点
- [ ] 推理链引用原文具体位置（段落/句子）
- [ ] 推理链展示完整推理过程
- [ ] 排除链逐一分析所有3个错误选项
- [ ] 排除链指出每个干扰项的陷阱类型
- [ ] 考点一句话格式正确、内容准确

### 常见扣分项
- 推理链未引用原文具体位置（-2）
- 排除链遗漏某个选项（-2）
- 推理链只说"根据原文可知"没有展示推理过程（-1）
- 考点一句话过于笼统（-1）

---

## 4. 干扰项合理性 (distractor_reasonableness)

### 评分标准
| 分数 | 标准 |
|------|------|
| 9-10 | 干扰项看似合理，陷阱类型清晰，学生容易误选 |
| 7-8 | 干扰项基本合理，个别陷阱不够隐蔽 |
| 5-6 | 部分干扰项明显错误，学生不会误选 |
| 1-4 | 干扰项荒谬，完全没有迷惑性 |

### 审核要点
- [ ] 每个干扰项与原文有某种关联（不是胡编的）
- [ ] 干扰项快速阅读时容易被误选
- [ ] 陷阱类型标注正确
- [ ] 同一题的3个干扰项使用不同陷阱类型

### 常见扣分项
- 干扰项与原文完全无关（-2）
- 干扰项明显错误，没有迷惑性（-2）
- 陷阱类型标注与实际不符（-1）
- 同一题两个干扰项使用相同陷阱（-1）

---

## 5. 收获有用性 (takeaway_usefulness)

### 评分标准
| 分数 | 标准 |
|------|------|
| 9-10 | 搭配高频实用，熟词生义精准，易错点有操作性 |
| 7-8 | 收获基本有用，个别搭配/词汇价值不大 |
| 5-6 | 搭配过于简单或过于偏僻，熟词生义牵强 |
| 1-4 | 收获内容无学习价值 |

### 审核要点
- [ ] 搭配来自文章，且具有学习价值
- [ ] 熟词生义的"常见义"确实是学生熟知的
- [ ] 熟词生义的"文中义"确实与常见义不同
- [ ] 易错点与具体题目关联
- [ ] 易错点的tip有操作性

### 常见扣分项
- 搭配过于简单（如make a decision）（-1）
- 熟词生义牵强，文中义和常见义区别不大（-2）
- 易错点tip空洞（如"要仔细阅读"）（-1）
- 搭配/词汇不是文章中的（-3）
