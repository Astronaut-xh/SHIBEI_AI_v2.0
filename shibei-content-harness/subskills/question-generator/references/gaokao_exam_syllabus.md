# 高考英语阅读理解考试大纲

## 考试概述

### 考试时间与分值
- 全国卷（I/II/III卷）及新课标卷
- 阅读理解部分：40分（满分150分），占总分约27%
- 建议用时：35分钟

### 阅读文章规格
| 指标 | 要求 |
|------|------|
| 每篇词数 | 250-400词 |
| 阅读总量 | 4篇约1200-1400词 |
| 语篇类型 | 说明文、议论文、记叙文、应用文 |
| 生词率 | 不超过3%（超纲词有中文注释） |
| 来源 | 外刊为主（NYT, Guardian, Economist, SciAm, Newsweek等） |

## 阅读理解考查能力

### 6项核心能力

| 能力 | 描述 | 对应题型 |
|------|------|---------|
| 1. 细节理解 | 获取文中具体信息 | 细节题 |
| 2. 推理判断 | 根据已知信息推理 | 推理题 |
| 3. 词义猜测 | 根据上下文猜测词义 | 词义题 |
| 4. 主旨概括 | 理解文章主旨要义 | 主旨题 |
| 5. 观点态度 | 理解作者观点态度 | （常融入推理题） |
| 6. 篇章结构 | 理解文章结构逻辑 | （常融入主旨题） |

## 命题惯例

### 题干规范
- 题干必须基于文章内容，不得依赖外部知识
- 题干表述清晰，无歧义
- 每题4个选项，形式一致

### 选项设计原则
- 正确选项必须有原文依据（同义改写或合理推断）
- 干扰项必须与原文有某种关联（不能完全无关）
- 选项长度大致相当
- 正确选项不应是最长或最短的
- 选项之间不互相排斥（除判断题外）

### 不宜考查的内容
- 纯粹的日期、数字记忆（需附带理解）
- 需要专业背景知识的信息
- 个人观点题（作者态度除外）
- 超出文本范围的过度推断

## 评分标准

### 客观题评分
- 每题2分（全国卷），共20题
- 答对得分，答错不扣分

## 文章体裁特征

### 说明文
- 客观陈述事实、数据、研究发现
- 常见于科技、环境、健康类话题
- 行文逻辑：现象→原因→影响→对策

### 议论文
- 呈现观点和论证过程
- 常见于社会、经济类话题
- 行文逻辑：论点→论据→结论

### 新闻报道
- 开头概括事件核心（5W1H）
- 按重要性递减排列信息

## 高考与四六级难度对照

| 级别 | 词汇量 | 阅读速度 | 文章来源 |
|------|--------|---------|---------|
| 高考 | 3500词 | 60-70 wpm | 外刊简写版 |
| CET-4 | 4500词 | 80-100 wpm | 外刊原文节选 |
| CET-6 | 6000词 | 100-120 wpm | 外刊原文 |
